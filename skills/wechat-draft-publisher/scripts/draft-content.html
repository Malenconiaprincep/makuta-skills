平稳小日记｜在家测血压，怎样才算「测对了」？

摘要：本文面向日常在家记录血压的用户，整理测量姿势、记录习惯与读数解读，便于在小程序里形成稳定、可比的数据。

一、为什么「同一时间、同一姿势」很重要？

血压在一天里会波动。若有时晨起、有时睡前才测，数字之间不好对比。建议固定：

  • 时段：例如每天起床后 1 小时内、服药前（若遵医嘱服药）。
  • 环境：安静坐定 5 分钟后再测，避免刚爬楼、喝咖啡、吸烟后立即测。
  • 姿势：背部有靠、双脚平放地面，袖带与心脏大致同高。

二、家庭血压「大致参考」（仅供科普，个体以医生为准）

分类 收缩压（高压） 舒张压（低压） 备注
正常高值 120–139 和/或 80–89 建议改善生活方式并规律监测
高血压 1 级 140–159 和/或 90–99 需医生评估是否用药
高血压 2 级 ≥160 和/或 ≥100 务必就医随访

⚠️ 表格为常见科普口径，不能替代诊断。合并妊娠、肾病、糖尿病等情况时，目标值可能不同。

三、一周记录示例（虚构数据，测表格与排版）

下面模拟用户在「平稳小日记」里连续记录一周（早晚各一次）：

日期 时段 收缩压 舒张压 心率 备注
周一 晨起 128 82 72 睡眠一般
周一 睡前 118 76 68 散步 20 分钟
周二 晨起 132 84 74 赶时间略紧张
周二 睡前 120 78 70 —
周三 晨起 126 80 71 —
周三 睡前 116 74 67 —

小技巧：若某次明显偏高，可在安静休息 3～5 分钟后复测一次，以第二次为准更稳妥（仍以医生建议为准）。

四、测量前检查清单（无序列表）

  • [ ] 静坐 ≥5 分钟
  • [ ] 袖带松紧适中（能塞进 1～2 根手指）
  • [ ] 手臂自然平放桌面，不要握拳说话
  • [ ] 连续测 2 次时，间隔 1～2 分钟

五、小程序里「结构化记录」长什么样？（代码块）

若后端或本地缓存用 JSON 存一条记录，可能是下面这种结构（示例）：

{
  "id": "bp_20250327_0730",
  "measuredAt": "2025-03-27T07:30:00+08:00",
  "systolic": 126,
  "diastolic": 80,
  "pulse": 71,
  "arm": "left",
  "tags": ["晨起", "服药前"],
  "note": "昨夜睡眠 6h"
}

前端展示时可能会做简单格式化（伪代码）：

function formatBp(entry) {
  const { systolic, diastolic, pulse } = entry;
  return `${systolic}/${diastolic} mmHg · 心率 ${pulse ?? "—"} 次/分`;
}

九、术语小抄

  • 收缩压：心脏收缩时血管内的压力，俗称「高压」。
  • 舒张压：心脏舒张时的压力，俗称「低压」。
  • 家庭自测 往往比诊室测略低，不要直接拿家庭数值去套诊室的单一阈值，交给医生综合判断。

文末一句：把测量变成习惯，比追求单次数字更重要 —— 平稳小日记 陪你慢慢把曲线走平。
